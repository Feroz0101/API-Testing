## 1) Run API Tests in Postman

### Step A — Import Collection & Environment
1. Open **Postman**
2. Go to **Import**
3. Import:
   - Postman Collection: `Student API testing`
   - Postman Environment: `Student ENV` :contentReference[oaicite:9]{index=9}  

> Tip: Keep your base URL and any IDs as variables in the environment.

### Step B — Run Requests Manually
1. Select the environment: **Student ENV**
2. Open any request → click **Send**
3. Verify:
   - Status code (e.g., Create Student returns **201 Created**) :contentReference[oaicite:10]{index=10}
   - Response body fields (id, first_name, etc.) :contentReference[oaicite:11]{index=11}

### Step C — Add Tests (Assertions) in Postman
In each request, go to the **Tests** tab and add assertions like:

```js
pm.test("Status code is 200", function () {
  pm.response.to.have.status(200);
});
